Project Title: 
Digital Personal Recipe Tracker

How to run my program:
Once program is ran, you will be provided with a list of prompts numbered 1-5. The user will input a number into the program to run the option they wish to use. Options include add recipe, search for recipe, list stored recipes, and print recipes. Upon selection, the user will follow the instructions the program guides them to do. Upon satisfation, the user will end the program by going to the main menu and exiting the program.

Short summary/features:
Within the Personal Recipe Tracker App, you will be presented a menu list of features: 
1. Add recipe.
    In this feature, you will be able to insert the ingredients, quanities, units, and recipe instructions for any recipe you wish. 
2. Search for recipe. 
    You will be given the option to search for any stored recipes that have been previously put into the program. Upon searching, if the specific recipe is store, the whole recipe will be printed. If they don't exist yet, will be brought back to the main menu to add it. 
3. List stored recipes.
    Upon this selection, you will be given a list any current stored recipes you have.
4. Print Recipes.
    This option prints the full recipe name, ingredients, and units for every stored recipe you have input into the program.
5. Exit. 
    This exits the program, you select this upon satisfaction.
